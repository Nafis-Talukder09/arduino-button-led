# Arduino Button LED Project

This is my second Arduino project.

## What it does
- LED turns ON when button is pressed
- LED turns OFF when button is released

## Components
- Arduino UNO
- LED
- 220 ohm resistor
- Push button

## Concepts learned
- Digital input/output
- INPUT_PULLUP
- Basic if-else logic
